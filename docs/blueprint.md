# **App Name**: CleanOps Hub

## Core Features:

- Interactive Dashboard UI: Modernize the existing tabbed interface for sites, cleaners, and schedules into responsive Next.js components.
- Site Management: Allow administrative users to add, edit, and view details for all cleaning sites, including their current status.
- Cleaner Management: Enable managing cleaner profiles, including their names and performance ratings.
- Cleaning Schedule Editor: Provide a dedicated interface to create, update, and visualize cleaning schedules, linking sites to cleaners and specified time slots.
- Real-time Status Updates: Facilitate quick updates to site statuses and cleaner performance ratings through interactive dropdowns, with data persistence.
- Risk Assessment Overview: Automatically categorize and display sites as 'Green' (good), 'Amber' (monitor), or 'Red' (risk) based on their updated statuses.
- AI-Powered Daily Operations Report Tool: Generate a natural language daily report summarizing site operations, highlighting trends and sites requiring attention based on current data, acting as an intelligent assistant.

## Style Guidelines:

- Light color scheme, built around professionalism and efficiency. The primary color, a deep, modern blue-grey (#3D4653), provides a stable and authoritative feel. The background is a very light desaturated blue-grey (#F1F3F6), offering a clean and airy canvas. A vibrant green accent color (#23C363) highlights active elements and success states, maintaining a connection to the original dashboard's positive indicators.
- Body and headline font: 'Inter' (sans-serif) for its clear, objective, and modern aesthetic, ensuring high readability for data-rich tables and reports across all screen sizes.
- Use clean, minimalist outline icons for actions and navigation, enhancing clarity and maintaining a contemporary professional appearance. Icons should be easily understandable at a glance.
- A responsive, grid-based layout for all dashboard sections, ensuring content is well-organized and scales gracefully across various devices. Emphasize ample whitespace to reduce visual clutter and improve data comprehension.
- Incorporate subtle, fast-paced transitions for tab switching and data filtering. Animations should feel crisp and responsive, improving user feedback without distracting from the operational data.
